# Role
Act as a Senior React Frontend Developer specializing in document generation.

# Objective
Create a production-ready PDF Report Component (`SanadReportPDF`) for the "Sanad" Career Coaching application.
The PDF must be visually stunning, formatted as a professional A4 document, and strictly support **Arabic Language (RTL)** without rendering issues (disconnected letters).

# 1. Technical Stack & Setup
- **Library:** `@react-pdf/renderer` (Latest version).
- **Fonts:** You MUST use a custom Arabic font (e.g., "Cairo" or "Tajawal") to ensure glyphs render correctly. Standard fonts will fail.

# 2. Implementation Steps
